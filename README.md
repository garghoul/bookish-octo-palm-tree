This simulator is to be a basic simulator of Magic the Gathering Commander game that can be ran with the use of Visual Studio.

This program will create three opponents to be played against. At this level, the opponents can only do simple actions such as attack with creatures and play removal spells. 
As the turn count gets higher and higher, the opponents will be capable of doing more powerful actions.

The purpose of this simulator is to test decks that to see if they are capable of dealing with all the basic scenarios that could happen in a game of commander.
